# Digital Clock ⏰

A sleek, modern digital clock built with HTML, CSS, and JavaScript featuring a beautiful gradient background, glowing effects, and responsive design.

## Features ✨

- **Real-time Display**: Shows current time in HH:MM format with separate seconds display
- **Date Information**: Displays full date (Month Day, Year) and day of the week
- **Visual Effects**: 
  - Glowing neon-style text with shadows
  - Blinking colon separator
  - Glassmorphism design with backdrop blur
  - Smooth hover animations
- **Responsive Design**: Adapts seamlessly to different screen sizes
- **Modern UI**: Clean, minimalist interface with a blue gradient background

## Technologies Used

- **HTML5**: Structure and semantic markup
- **CSS3**: Styling, animations, and responsive design
- **JavaScript**: Real-time clock functionality and DOM manipulation

## Installation & Usage

1. **Clone or Download**: Save the HTML file to your local machine
   
2. **Open the File**: Simply open `index.html` (or your filename) in any modern web browser

3. **That's it!** The clock will start running automatically

No build process, dependencies, or server required!

### Time Format
To change to 12-hour format with AM/PM, modify the JavaScript `updateClock()` function.

## Responsive Breakpoints

- **Desktop**: Full-size display (4rem clock font)
- **Mobile** (≤768px): Scaled-down version (2.5rem clock font)
